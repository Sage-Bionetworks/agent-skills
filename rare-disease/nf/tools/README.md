# NF Research Tools Skills

Skills for working with the [NF Research Tools Central](https://www.synapse.org/Synapse:syn26338068/tables/) — a catalog of 9 NF research resource types (antibodies, cell lines, animal models, computational tools, etc.) managed via the [nf-research-tools-schema](https://github.com/nf-osi/nf-research-tools-schema) repo.

| Skill | Description |
|---|---|
| `schema/` | Data model: CSV source of truth, JSON-LD generation, JSON form schemas, submission pipeline |
| `synapse-tables/` | Querying and updating the Synapse database (table IDs, SQL patterns, upsert, snapshots) |
| `workflows/` | GitHub Actions pipeline: publication mining with Claude AI, schema updates, Synapse upserts |
