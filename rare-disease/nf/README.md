# NF-OSI Agent Skills

Skills for the NF-OSI (Neurofibromatosis Open Science Initiative) data management and curation stack. Organized by the repo or system each skill covers.

| Directory | Repo(s) | Description |
|---|---|---|
| `tools/` | [nf-research-tools-schema](https://github.com/nf-osi/nf-research-tools-schema) | Schema, database tables, and workflows for the NF Research Tools Central catalog |
| `metadata-dictionary/` | [nf-metadata-dictionary](https://github.com/nf-osi/nf-metadata-dictionary) | LinkML schema authoring, term contribution, and Synapse schema registration |
| `projects/` | [dcc-site](https://github.com/nf-osi/dcc-site) | DSP intake, Synapse project provisioning, checkpoints, data release, Jira support |
| `portal/` | — | NF Data Portal Synapse tables: file view annotations, search facets, portal utilities |
| `agent/` | [mcp-server](https://github.com/nf-osi/mcp-server), [agent-at-work](https://github.com/nf-osi/agent-at-work) | nfty MCP server setup, portal dataset curation recipes, automated curation task generation |
