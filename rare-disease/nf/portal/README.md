# NF Portal Skills

Skills for working with NF Data Portal Synapse tables — querying and managing file-level annotations, configuring search facets, and generating portal reports.

| Skill | Description |
|---|---|
| `annotations-file-view/` | NF Portal file view [syn52702673](https://www.synapse.org/Synapse:syn52702673/tables/): querying, bulk annotation updates, search facet configuration, schema binding to folders, curation task creation |
| `nfportalutils/` | R package for NF portal project setup (`new_project`, `register_study`) and checkpoint report generation via Docker |
