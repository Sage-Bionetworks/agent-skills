# NF Metadata Dictionary Skills

Skills for working with the [nf-metadata-dictionary](https://github.com/nf-osi/nf-metadata-dictionary) — the LinkML-based schema that defines controlled vocabularies, data templates, and submission forms for all NF Data Portal metadata.

| Skill | Description |
|---|---|
| `metadata-dictionary/` | LinkML YAML module authoring, term contribution guidelines, artifact generation pipeline (JSON-LD, JSON schemas), Synapse schema registration, and platform limits |
