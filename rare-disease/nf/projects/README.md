# NF Projects Skills

Skills for managing the lifecycle of NF research projects, from initial data sharing plan (DSP) submission through Synapse project provisioning, active management, and data release. Powered by the [dcc-site](https://github.com/nf-osi/dcc-site) repo and tracked on the [GitHub project board](https://github.com/orgs/nf-osi/projects/1).

| Skill | Description |
|---|---|
| `provisioning/` | DSP intake → review validation (`review.py`) → Synapse project creation → document generation and PI notification |
| `lifecycle/` | Post-provisioning: 30-day checkpoint automation (grant-end, data-release), checklists, GitHub project board, ProjectLive sync |
| `atlassian/` | Jira service desk triage (NFOSI queue) and Confluence documentation using the Atlassian MCP server |
