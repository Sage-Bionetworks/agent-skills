# Agent Skills
This repository contains agent skills leveraged across Sage Bionetworks. Each skill is a `SKILL.md` file providing an AI agent with the context, code patterns, and institutional knowledge needed to work in a specific system or repo without re-explanation.

## Structure

```
synapse-python-client/     General-purpose Synapse Python client
rare-disease/              Skills for rare disease programs
```

## Contributing

Add a new skill by creating a `SKILL.md` in the appropriate subdirectory. Place it under the directory that best matches the repo or system it covers. Add a `README.md` to the parent directory if one doesn't exist yet.
