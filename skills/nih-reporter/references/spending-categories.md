# RCDC Spending Category IDs (FY2024)

Numeric IDs for the `spending_categories.values` criteria parameter. Pass as integers:

```json
"spending_categories": {"values": [27, 105, 132], "match_all": false}
```

`match_all: true` returns projects categorized in **all** of the listed IDs (intersection). `match_all: false` returns projects in **any** (union — usually what you want for "give me everything cancer-related").
